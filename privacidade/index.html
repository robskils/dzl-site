Política de Privacidade

Como o Dojo Zen de Lisboa trata os dados pessoais dos utilizadores deste website.

Última actualização: Maio de 2026

1. Responsável pelo tratamento

O responsável pelo tratamento dos dados pessoais recolhidos através deste website é o Dojo Zen de Lisboa – Ryumonji, contactável através do endereço electrónico contacto@dojozenlisboa.com.

2. Dados recolhidos

Este website recolhe dados pessoais exclusivamente quando o utilizador os fornece voluntariamente através do formulário de contacto. Os dados recolhidos são:

  • Nome
  • Endereço de correio electrónico
  • Número de telefone (opcional)
  • Assunto e mensagem enviados

Não recolhemos dados de forma automática através de cookies de rastreamento ou ferramentas de análise de terceiros.

3. Finalidade do tratamento

Os dados pessoais recolhidos são utilizados exclusivamente para responder às questões e pedidos submetidos pelo utilizador através do formulário de contacto. Não são utilizados para fins comerciais, não são partilhados com terceiros e não são utilizados para envio de comunicações não solicitadas.

4. Base legal

O tratamento dos dados baseia-se no consentimento do utilizador, expresso pelo acto de submissão voluntária do formulário de contacto.

5. Conservação dos dados

Os dados são conservados apenas pelo período necessário para responder ao pedido do utilizador, após o qual são eliminados.

6. Direitos do utilizador

O utilizador tem o direito de, a qualquer momento:

  • Aceder aos dados pessoais que nos forneceu
  • Rectificar dados incorrectos ou incompletos
  • Solicitar a eliminação dos seus dados
  • Retirar o consentimento dado
  • Apresentar reclamação à autoridade de controlo (CNPD — Comissão Nacional de Protecção de Dados)

Para exercer qualquer destes direitos, contacte-nos através de contacto@dojozenlisboa.com.

7. Segurança

Este website é servido exclusivamente por HTTPS. Os dados submetidos pelo formulário de contacto são transmitidos de forma encriptada.

8. Alterações a esta política

Reservamo-nos o direito de actualizar esta política de privacidade. Qualquer alteração significativa será comunicada através deste website.

9. Contacto

Para qualquer questão relacionada com a privacidade dos seus dados, contacte-nos:

Dojo Zen de Lisboa – Ryumonji
contacto@dojozenlisboa.com
